# Prompt de Documentación

<tarea>

Genera o actualiza el archivo README.md del proyecto.

</tarea>

<objetivo>

Documentar la solución refactorizada de manera que cualquier desarrollador pueda:

* Comprender la arquitectura.
* Ejecutar el proyecto.
* Ejecutar las pruebas.
* Validar manualmente el comportamiento.

</objetivo>

<contenido_obligatorio>

1. Descripción del proyecto.
2. Problema original del código legacy.
3. Arquitectura implementada.
4. Estructura de carpetas.
5. Responsabilidades de:

   * TaxCalculator
   * PaymentProvider
   * OrderRepository
   * ProcessOrderUseCase
6. Instalación.
7. Scripts disponibles.
8. Ejecución local.
9. Ejecución de pruebas.
10. Procedimiento de validación manual.
11. Decisiones técnicas.
12. Consideraciones de seguridad.
13. Posibles mejoras futuras.

</contenido_obligatorio>

<restricciones>

* No inventar scripts inexistentes.
* No documentar funcionalidades no implementadas.
* Mantener el README en español.
* Utilizar ejemplos reales del proyecto.

</restricciones>

<resultado_esperado>

Generar el contenido completo del archivo README.md listo para ser guardado en la raíz del proyecto.

</resultado_esperado>
