# Prompt de Seguimiento

<tarea>

Actúa como un Software Architect Senior y realiza una revisión técnica de la solución propuesta anteriormente.

</tarea>

<validaciones>

Revisar:

* Separación de responsabilidades.
* Cumplimiento de Clean Architecture.
* Seguridad.
* Escalabilidad.
* Mantenibilidad.
* Manejo de errores.
* Acoplamiento.
* Testeabilidad.

</validaciones>

<preguntas>

1. ¿Existen dependencias innecesarias?
2. ¿Hay riesgos de seguridad?
3. ¿La lógica de negocio está correctamente aislada?
4. ¿Existen posibles errores en tiempo de ejecución?
5. ¿Hay oportunidades de simplificación?
6. ¿La solución soporta nuevos países y nuevos proveedores de pago?

</preguntas>

<resultado_esperado>

Entregar:

* Hallazgos.
* Nivel de riesgo de cada hallazgo.
* Recomendaciones.
* Código corregido únicamente si es necesario.

</resultado_esperado>
